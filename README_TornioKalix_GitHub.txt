This archive contains R code and data files necessary to perform the analyses detailed in the manuscript "A large wild salmon stock shows genetic and life history differentiation within, but not between, rivers" (Miettinen et al., Conservation Genetics).

Folder 'TornioKalix_Msat_R_Codes': R files containing commented code used for running statistical analyses.

Folder 'TornioKalix_Msat_Data': input files with data needed for performing the analyses in the manuscript.
